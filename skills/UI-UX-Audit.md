# UI-UX-Audit

このスキルは、WebサイトやアプリケーションのUI/UXを評価し、改善案を提示するために使用されます。
UIコンポーネントの使い勝手、アクセシビリティ、パフォーマンス、デザインの一貫性をチェックし、
`audit`（Lighthouseなど）ツールと連携して具体的な改善レポートを作成します。

## 指示
1. 対象となるページまたはコンポーネントを特定する。
2. ユーザビリティの観点（直感性、操作手順の少なさなど）から評価を行う。
3. パフォーマンスやアクセシビリティのボトルネックがないか確認する。
4. 具体的なコード修正案やデザイン改善案を提示する。
